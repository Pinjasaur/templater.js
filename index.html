templater.js

A tiny JS library for minimal {{templating}}.

Download Script → View on GitHub →

tl;dr

templater.js is a teeny JavaScript library for minimal templating needs (soon to be made obsolete by ES6 templating). Its templating expression syntax mimics that of mustache/Handlebars. Barebones conditional logic included.

Features

  • No evals or new Function() { … }s
  • HTML characters (<, >, and &) automatically escaped
  • < 1KB after minification
  • Zero dependencies

Limitations

  • Cannot nest logic statements
  • Cannot nest properties (e.g. {{foo.bar.baz}}

Getting Started

  1. Add the script to your HTML document: 
    <script src="path/to/templater.min.js"></script>
  2. Create the template you'd like to use:
    3. <!-- by using a non-standard `type` the browser won't parse it -->
<script id="my-template" type="text/template">
  <h2>Hi, my name is {{ name }}.</h2>
  <p>I am {{ age }} years old and live in {{ location }}.</p>
</script>
  3. Parse the template against some data and render it to the DOM: 
    var template = templater(document.getElementById("my-template").innerHTML);

document.body.insertAdjacentHTML("beforeend", template({
  name: "John",
  age: 27,
  location: "The United States"
});
